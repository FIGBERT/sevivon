# DEPRECATED: Sevivon

**Sevivon** is an open-source multiplayer [dreidel][dreidel] game for
mobile devices, built with [Godot][godot]. This repository was an
early prototype, and has been abandoned. See [here][sevivon] for the
new repo.

[dreidel]: https://en.wikipedia.org/wiki/Dreidel
[godot]: https://godotengine.org/
[sevivon]: https://git.figbert.com/FIGBERT/Sevivon
